CommandTracker
==============

CommandTracker is a simple utility plug-in that tracks the command you run.

Building Sample
--------------------
To build the sample, you are going to need:

* Rhinoceros 7 (http://www.rhino3d.com)
* Microsoft Visual C# 2019
* Microsoft .NET Framework 4.8
